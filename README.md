# hashfile

EXAMPLE/TESTING PROJECT!
wrapper around:
```
$ cd (dirname FILENAME)
$ rhash -o FILENAME.sha1 --sha1 FILENAME.zip
```

## Install

With [fisherman]

```
fisher f4bio/hashfile
```

## Usage

```fish
hashfile
```
